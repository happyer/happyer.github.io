项目经验

​ 车百用App

​ 提供一站式汽车管家服务平台，目前app中包含了车前销售，售后保养维修等一站式的服务，其中还涉及到了紧急救援及专家问答模块。

​ 整个使用Maven工具来构建，项目结构上主要分为表现层、业务层和数据访问层，层次间的依赖关系自下到上。采用的技术有SpringMVC,Spring,Mybatis,Spring AOP, Maven,Quartz 等。其中表现层采用SpringMVC 框架开发以及使用Velocity作为试图的渲染同时使用restful 形式提供给app端以及后台管理系统使用了；业务层封装业务流程，为适应业务的变更，每一业务模块均有专门的接口及实现类，利用Spring的IoC功能将实现类注入给表现层的Controller；数据访问层借助于Mybatis实现,代码简洁且可适应不同的数据库。事务部分利用Spring的声明式事务管 理。使用Spring AOP作为日志记录和APP的异步推送，通过Quartz 来执行一些定时的清理作业。以及使用全文检索引擎Solr提供APP 的问答模块的搜索业务。为提高性能，系统的响应时间,使用了nginx+jetty负载均衡同时，将一些静态的数据使用了云存储，以及数据库的查询缓存。

​ 后台管理系统

​ 后台管理系统主要是针对使用智能家居的用户硬件设备的管理

​ 该项目主要对使用智能家居的用户设备进行管理，使用MVC设计模式，其中视图控制器使用Struts框架开发，jquery bootstrap前段的设计与布局；service封装业务流程，为适应业务的变更，每一业务模块均有专门的接口及实现类，利用Spring的IoC功能将实现类注入给表现层的Action；数据访问借助于Mybatis实现。同时提供了APP端与数据采集端视频流的转发服务，在网络的对称结构的情景下，使用了Google 的turn server 进行了视频数据流的转发。
